== Age & Experience Guide

[horizontal]
Ages 8-12, No Experience:: **Me Dialect** - Learn fundamental concepts with visual, safe syntax
Ages 8+, After Me:: **Solo Dialect** - The foundation. Many stay here for life
Undergrad+, Need AI:: **Duet Dialect** - When AI assistance adds value to your work
Industry/Advanced:: **Ensemble Dialect** - When orchestrating AI agent teams

== Important Notes

=== Solo is Complete

**Solo is not a stepping stone.** It's a fully capable systems programming language. Many professional developers will use Solo exclusively and never need Duet or Ensemble. Only move to the advanced dialects if your specific application requires it:

- **Duet**: Your work involves AI-assisted code generation
- **Ensemble**: You're building multi-agent AI applications

=== Progressive, Not Linear

You don't have to use every dialect in order. The progression is:

1. **Me** → Learn concepts (required foundation)
2. **Solo** → Build things (where most people stay)
3. **Duet/Ensemble** → Specialized tools (use if needed)

== Getting Started

=== For Absolute Beginners (Ages 8-12)

[source,bash]
----
# Start with Me
git clone https://github.com/hyperpolymath/me-dialect-playground.git
cd me-dialect-playground
deno task demo
----

=== For Programmers Ready to Build

[source,bash]
----
# Clone this repo for Solo/Duet/Ensemble
git clone --recursive https://github.com/hyperpolymath/mylang-playground.git
cd mylang-playground/hives/solo
----

== Philosophy

[quote]
Learn what you need, when you need it. The best dialect is the one that fits your current needs.

- **Me** teaches you to think like a programmer
- **Solo** lets you build real applications
- **Duet** amplifies your productivity with AI
- **Ensemble** unlocks AI-native architectures

Choose your path. There's no wrong answer.
